# HTML Viewer Live

Live HTML playground to write and preview HTML, CSS, and JavaScript in real time.

## 🚀 Features

- Real-time HTML rendering
- Isolated preview using iframe
- Simple and fast UI
- No dependencies
- Runs entirely in the browser

## 🌐 Demo

https://CarlosOrtiz.github.io/html-viewer/

## 📦 Usage

1. Write your HTML in the editor
2. See instant results on the right panel

## ⚠️ Security Note

This tool renders raw HTML inside an iframe. Avoid running untrusted code if deployed publicly.
